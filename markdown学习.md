# 列表类
## 序号列表
### 有序列表：数字+点+空格
1. 序列1
2. 自动生成如下序列

### 无序列表：-、*、+ 中的任意符号，后加空格
- 用-创建的序列
* 用*创建的序列
+ 用+创建的序列

==若是要在无序列表下创建分列表，则在符号前加两个空格即可。==

- wuxu
  - wuxu
- wuxu
  - wuxu

## 任务列表
未完成任务：“-”+空格+[ ]
已完成任务：在方括号里加“x”

如：
- [ ] 这是一个未完成任务
- [x] 这是一个已完成任务

## 引用 
### 单行引用
单行引用使用“>”符号完成
如：
>人生如逆旅我亦是行人
>苏轼

### 多层嵌套引用
使用多个“>"完成。
>一层
>>two
>>>three
